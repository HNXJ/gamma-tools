# Agent Role Map

- Back Agents: Execution, harnesses, receipts, runtime validation.
- Front Agents: Observation/browser validation only (gamma-arena).
- Analysis Agents: Reports, benchmarks, evidence derivation.
- Doctrine Agents: Prompts, audits, plans, validation criteria.
- Claude: CLI/back or teacher/high-level roles.
- Gemini CLI: CLI/back, doctrine, alignment work.
- Antigravity: Front/browser validation (restricted to gamma-arena).
- Guard Agents: Structured-output safety, admission, drift checks, no Truth mutation.
- gamma-tools: Alignment hub only, not runtime or truth authority.
