# Agent Commons Charter

Governing Rule:
gamma-tools is free for agent collaboration only when content is Gamma-protocol compliant, science/reasoning relevant, no-secret, no unsafe runtime behavior, and not Truth-plane authority.

Allowed Content:
- Reusable prompts, coordination templates, and routing artifacts.
- Agent operating notes and alignment repair checklists.
- Cross-agent handoff templates.
- Safety and risk audit documentation.

Forbidden Content:
- Secrets, credentials, keys, bearer tokens, passwords.
- Live Truth-plane state or unverified scientific results.
- Unsafe runtime authority (shell, network, remote commands).

Compliance:
- GitHub Project gamma is the durable Control ledger.
- Respect strict plane separation (Control, Execution, Truth, Observation).
